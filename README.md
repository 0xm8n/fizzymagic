# required
- nodejs (https://linuxize.com/post/how-to-install-node-js-on-ubuntu-20-04/)

# install dependencies
```
npm in
```

# build and test
## build
```
npm run build
```

## test
```
npm run test
```

## start localhost testnet
```
npm run node
```

### deploy to localhost testnet
```
yarn deploy
```
